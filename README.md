# DevCraft

Interaktive Lern- und Referenzplattform für professionelle Softwareentwicklung.

## Inhalt

15 Kernthemen · 75 Subbereiche · 30 Deep Dives

| Thema | Thema | Thema |
|---|---|---|
| Software Development Process | Git & Versionskontrolle | Softwarearchitektur |
| Testing | Security | Docker & Container |
| CI/CD | Datenbanken | APIs |
| Monitoring | ... | |

Jedes Thema enthält:
- **Überblick** – Einordnung und Kontext
- **Subbereiche** – Konzept, Key Points, Tools, Commands
- **Deep Dives** – Code-Beispiele, Diagramme, Best Practices, häufige Fehler

## Starten

Kein Build-Tool, kein Server, keine Installation nötig.

```
index.html per Doppelklick im Browser öffnen
```

## Technologie

Reines HTML/CSS/JavaScript – keine Abhängigkeiten, keine Frameworks.

- **Lucide Icons** via CDN
- **Google Fonts** (Inter, JetBrains Mono) via CDN
- Hash-basiertes Routing (`#/topic/subarea`)
- Dark Theme Design System mit CSS Custom Properties
